# Analyze Resume powered - AI


---

## 🛠 Tech Stack
* **Frontend**: Next.js (React), Tailwind CSS
* **Backend**: Next.js API Routes, Prisma (ORM)
* **Database**: PostgreSQL (Supabase)
* **Communications**: Resend API
* **Deployment**: Vercel

## ✨ Key Features
* **Automate Sourcing**: Curates openings from credited sources to ensure high-signal application opportunities.
* **Filter Granularly**: Targets specific roles including **SWE, ML, Data, Frontend, and Backend**.
* **Deliver Automated Digests**: Sends scheduled weekly briefings and instant notifications via **Resend API**.
